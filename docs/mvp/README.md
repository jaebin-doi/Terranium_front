# MVP Documentation

This folder contains MVP scope and implementation planning documents.

Current documents:

- `PRODUCT_MVP_SCOPE.md`: MVP-A scope, workflows, screens, domain model, and first implementation order.
- `PHASE_0_TECHNICAL_POC_PLAN.md`: Phase 0 technical PoC implementation plan.
- `phase-0/`: Step-level Phase 0 implementation plans.

Rules:

- Detailed implementation plans in this folder should be written in English.
- Completed implementation or verification work must create a Korean progress report under `docs/reports/`.
- Korean progress reports are written by Claude after implementation or verification is complete, not during planning.
- Keep unresolved product or implementation questions explicit.
